# Themen – Bilder

Hier können Sie themenspezifische Fotos ablegen. Die Seiten verwenden standardmäßig `placeholder.webp`.

**Optionale Dateinamen** (werden von den jeweiligen Themen-Seiten genutzt, sobald vorhanden):

- Textilthemen: `t-shirts-bedrucken.webp`, `schuerzen-besticken.webp`, `kappen-veredeln.webp`, usw. (Slug = Dateiname ohne .webp)
- Sportbekleidung: `sportbekleidung-bedrucken/laufshirts-bedrucken.webp`, usw.
- Arbeitskleidung: `kasacks-besticken-medizin-bekleidung.webp`

Format: WebP (oder JPG/PNG), Querformat 4:3 oder 16:9 empfohlen.
