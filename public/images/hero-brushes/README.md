# Hero title brush SVGs

This folder holds 3 brush/highlighter SVGs used under each line of the hero card title.

## Free brush SVG libraries (for more or alternate assets)

- **SVG Repo** – https://www.svgrepo.com/  
  - [Highlighter Underline](https://www.svgrepo.com/svg/267876/highlighter-underline) (CC0)  
  - [Brush Stroke](https://www.svgrepo.com/svg/13981/brush-stroke) (CC0)  
  - [Highlighter Draw](https://www.svgrepo.com/svg/281254/highlighter-draw) (CC0)  
  - [More highlighter underlines](https://www.svgrepo.com/vectors/highlighter-underline/)

- **Vecteezy** – https://www.vecteezy.com/free-vector/marker-brush-stroke  
  Marker brush stroke vectors (SVG/PNG).

- **IconScout** – https://iconscout.com/icons/brush-stroke  
  Brush stroke icons and marker brush illustrations (SVG/PNG).

Download SVGs and place them here, then reference in `app.css` (e.g. `brush-1.svg`, `brush-2.svg`, `brush-3.svg`) if you replace the built-in ones.
